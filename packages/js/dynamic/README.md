# dynamic.js ⚡️

Working with LLMs has never been easier.

## Install

```bash
# npm
npm install

# yarn
yarn

# pnpm
pnpm install
```

## Testing

Currently, all features are tested via the example API app found [here](./../examples/api/).

## Linter

```bash
# run linter
npm run lint

# run linter and fix
npm run fix
```
